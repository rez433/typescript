Simple file structure:
in terminal: "tsc --init"
put "main.ts" in src folder
put all other public files in public folder
in "tsconfig.json", uncomment "outDir" and "roodDir" and give the path to respective file
"outDir": "./public",
"rootDir": "./src",
to avoid compiling other .ts files outside src folder, add '"include": ["src"]' to "tsconfig.json"
